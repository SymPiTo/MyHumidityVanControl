# MyHumidityVanControl

Dieses Modul steuert einen Lüfter (Van) basierend auf einem Feuchtigkeitsschwellwert.

### Funktionen
- Automatische Reaktion auf Variablenänderungen des Sensors via `MessageSink`.
- Einstellbarer Schwellwert.
- Debug-Ausgabe für einfache Fehleranalyse.

### Installation
1. Repository in IP-Symcon hinzufügen.
2. Instanz "MyHumidityVanControl" erstellen.
3. Sensor und Aktor in der Konfiguration wählen.
